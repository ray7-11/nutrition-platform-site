# 營養餐與生物數據平台網站

這是一個可直接部署到 GitHub + Vercel 的 React / Vite 網站專案。

## 本機開啟
```bash
npm install
npm run dev
```

## 打包
```bash
npm run build
```

## 上傳到 GitHub
1. 建立新的 GitHub Repository
2. 把此資料夾全部上傳
3. 到 Vercel 匯入 GitHub 專案
4. Framework 選擇 Vite
5. 直接 Deploy 即可

## 目前頁面內容
- 平台首頁
- 功能說明
- 營養餐商城展示
- 健康儀表板展示
- AI 餐點辨識說明
- 專家後台價值
- 商業模式 CTA
